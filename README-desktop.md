# noted Desktop

Esta carpeta ya contiene el sitio web de `noted` y ahora también incluye un envoltorio de escritorio con Electron.

## Uso

1. Abre la terminal en `/Users/rokck/Downloads/rokck.github.io`
2. Ejecuta `npm install`
3. Ejecuta `npm start`

## Crear app macOS

- Ejecuta `npm run dist`
- Esto generará un paquete para macOS usando `electron-builder`.
